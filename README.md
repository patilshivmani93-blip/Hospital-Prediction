# 🏥 Hospital Readmission Prediction System

## About
This project predicts whether a patient needs 
to be readmitted to hospital based on their 
medical history and health parameters.

## Dataset
- 500 patients data
- Age, Gender, Disease, Department
- Blood Pressure, Sugar Level
- Billing Amount, Stay Days

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## ML Models Used
- Random Forest Classifier
- Logistic Regression

## Results
- Random Forest Accuracy: ~85%
- Logistic Regression Accuracy: ~80%

## How to Run
pip install pandas numpy matplotlib seaborn sklearn
python hospital_prediction.py
